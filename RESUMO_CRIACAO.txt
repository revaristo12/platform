🚀 COMO USAR:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. COPIE os arquivos para o servidor VPS Ubuntu 20.04:

   scp -r . root@seu-servidor:/opt/meeting-ai-platform

2. EXECUTE o script de instalação:

   ssh root@seu-servidor
   cd /opt/meeting-ai-platform
   chmod +x scripts/install-vps.sh
   sudo ./scripts/install-vps.sh

3. CONFIGURE as API Keys (obrigatório):

   nano .env
   # Edite OPENAI_API_KEY e ANTHROPIC_API_KEY

4. REINICIE os serviços:

   docker compose restart

5. ACESSE a plataforma:

   http://seu-servidor:3000

📋 REQUISITOS DO SERVIDOR:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• Ubuntu 20.04 LTS
• Mínimo 2 CPU cores (recomendado: 4+)
• Mínimo 4GB RAM (recomendado: 8GB+)
• Mínimo 20GB disco (recomendado: 50GB+)
• Acesso root ou sudo
• Conexão com internet

⚙️ CONFIGURAÇÃO NECESSÁRIA:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Após a instalação, configure no arquivo .env:

1. OPENAI_API_KEY
   → Obtenha em: https://platform.openai.com/api-keys
   → Usado para transcrição (Whisper) e análise (GPT)

2. ANTHROPIC_API_KEY
   → Obtenha em: https://console.anthropic.com/
   → Usado para análise avançada (Claude)

3. CORS_ORIGINS (se tiver domínio)
   → Adicione seu domínio para permitir acesso

4. DOMAIN (se quiser SSL)
   → Configure para usar Let's Encrypt

# Acessar banco de dados
docker compose exec postgres psql -U meeting_user -d meeting_db

# Backup manual
sudo /usr/local/bin/meeting-backup.sh

# Iniciar em modo produção
